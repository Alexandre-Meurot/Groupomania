# Groupomania

**Une fois le dossier *groupomania* clôné :**

## **Depuis le sous-dossier *backend* :**

   * installer les dépendances : `npm install`
   * lancer le serveur backend : npm run start
   

## **Depuis le sous-dossier *frontend* :**

* installer les dépendances : `npm install`
* lancer le front : `ng serve`
